# Design Trends 2026

design-director 에이전트가 트렌드 전략 수립(trend_strategy) 시 참조하는 레퍼런스 문서.
트렌드를 위한 트렌드를 경계하고, 제품 맥락에 맞는 항목만 선별하여 적용한다.

## 주요 트렌드 목록

### AI-native UX
- 생성형 UI: 사용자 컨텍스트에 따라 인터페이스가 실시간으로 재구성되는 패턴
- 적응형 레이아웃: 사용 패턴을 학습하여 개인화된 화면 구조를 제공
- 개인화 인터페이스: 사용자별 워크플로우에 최적화된 UI 컴포넌트 배치

### Glassmorphism 2.0
- 반투명 레이어와 블러 효과를 활용한 깊이감 표현
- 배경과의 대비를 유지하면서 WCAG 2.2 접근성 기준 충족 필수
- 과도한 블러로 인한 가독성 저하 방지

### Bento Grid 레이아웃
- 비균일 그리드를 활용한 정보 계층화
- 카드 크기 차등으로 중요도를 시각적으로 표현
- 반응형 처리 시 그리드 재배치 전략 필요

### 다크 모드 퍼스트
- 라이트 모드를 파생으로 두고 다크 모드를 기본으로 설계
- 컬러 토큰 기반으로 두 모드 간 자동 전환 보장
- OLED 화면 최적화를 위한 순수 블랙(#000000) 활용

### 모션 퍼스트 스토리텔링
- 정적 와이어프레임보다 모션 프로토타입을 먼저 검토
- 스크롤 기반 스토리텔링으로 콘텐츠 흐름 강화
- `prefers-reduced-motion` 미디어 쿼리 필수 대응

### 감성적 인터페이스 (Emotional UI)
- 마이크로인터랙션으로 사용자 행동에 즉각적인 피드백 제공
- 브랜드 퍼스낼리티를 인터랙션 패턴에 반영
- 오류 상황에서도 긍정적 감성 경험 유지

## 적용 원칙

1. **선별 기준**: 트렌드가 제품의 핵심 사용자 가치를 강화하는 경우에만 적용
2. **접근성 우선**: 어떤 트렌드도 WCAG 2.2 기준을 타협하지 않음
3. **성능 검토**: 모션/블러 효과 적용 시 60fps 유지 가능 여부 사전 확인
4. **점진적 도입**: 한 번에 모든 트렌드를 적용하지 않고, 우선순위를 정해 단계적으로 반영

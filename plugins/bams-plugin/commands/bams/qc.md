## 소속 에이전트

| 에이전트 | 스킬 트리거 예시 |
|----------|-----------------|
| `bams-plugin:qa-strategy` | 테스트 전략, 케이스 설계, 품질 리스크 |
| `bams-plugin:automation-qa` | UI E2E 자동화, API 자동화, 자동화 안정화 |
| `bams-plugin:defect-triage` | 결함 분류, 재현 보장, 근본 원인 추적 |
| `bams-plugin:release-quality-gate` | 출시 준비 검토, 롤백 점검, 출시 후 모니터링 |

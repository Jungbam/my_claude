# Bams: Engineering Hub

> `references/dept-hub-protocol.md`를 Read하여 5단계 실행 흐름과 viz 계측을 따른다 (Pattern A).

## 소속 에이전트

| 에이전트 | 스킬 트리거 예시 |
|----------|-----------------|
| `bams-plugin:frontend-engineering` | UI 컴포넌트, 클라이언트 플로우, 프론트 품질 |
| `bams-plugin:backend-engineering` | API 계약, 도메인 로직, 데이터 정합성 |
| `bams-plugin:platform-devops` | IaC 관리, CI/CD 오케스트레이션, 관측성/인시던트 |
| `bams-plugin:data-integration` | 이벤트 트래킹, 외부 시스템 연동 |
